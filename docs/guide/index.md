# Getting Started

Welcome to Useful Optimizer! This guide will help you get started with using the library for your optimization problems.

## What is Useful Optimizer?

Useful Optimizer is a comprehensive Python library containing 100+ optimization algorithms for numeric problems. The algorithms are organized into logical categories, making it easy to find and compare different approaches.

## Who is this for?

- **Researchers** studying optimization algorithms and benchmarking
- **Data scientists** looking for hyperparameter optimization
- **Engineers** solving real-world optimization problems
- **Students** learning about different optimization techniques

## Supported Algorithm Categories

| Category | Description |
|----------|-------------|
| **Swarm Intelligence** | Nature-inspired algorithms based on collective behavior |
| **Evolutionary** | Population-based algorithms using biological evolution |
| **Gradient-Based** | Algorithms using gradient information |
| **Classical** | Traditional mathematical optimization methods |
| **Metaheuristic** | Problem-independent algorithmic frameworks |
| **Physics-Inspired** | Algorithms based on physical phenomena |
| **Social-Inspired** | Algorithms based on social behavior |
| **Probabilistic** | Probabilistic and Bayesian approaches |
| **Constrained** | Methods for constrained optimization |
| **Multi-Objective** | Algorithms for multi-objective problems |
