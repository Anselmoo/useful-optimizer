## Prerequisites

Before you begin, ensure you have:

- **Python 3.10, 3.11, or 3.12** installed
- Basic understanding of Python programming
- Familiarity with NumPy (helpful but not required)

---

## What's Next?

1. **Install** the library using your preferred method
2. **Follow** the quick start guide to run your first optimization
3. **Explore** the algorithm documentation to find the right optimizer for your problem
